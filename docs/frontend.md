# Frontend

El frontend Next.js usa servicios HTTP reales en `frontend/services`.

Variable:

```env
NEXT_PUBLIC_API_BASE_URL=https://localhost:5001/api
```

Pantallas conectadas: dashboard, activo fijo, usuarios, herramientas y checklist. Los servicios base para dimensiones y maestros tambien estan disponibles para completar vistas administrativas.

UI: se ajusto sidebar corporativo azul, fondos suaves, cards con glow y estados de carga/error.
